Move your plastic cards to your smartphone with mTasku. Use the universal app to pay in a regular or online store and travel on public transport or as a replacement for a digital employment certificate – the possibilities are endless.

== Description ==

mTasku is [a smartphone application](https://www.mtasku.ee/en/) that allows to make payments with any registered payment card.
The plugin makes online payments possible using mTasku application.
In order to use the services you need to [sign up and apply for a contract](https://www.mtasku.ee/en/sooviavaldus/).
After that you need to configure the plugin by entering merchant credentials given to your shop.

mTasku on mobiilne rahakott, mis koondab endasse makse- ja kinkekaarte, laia valiku kliendikaarte, heategevuslikke annetuskaarte, transpordikaarti (Ühiskaart) ja palju muud.

E-poes mTaskuga maksmine on kiire ja mugav, kuna kliendid ei pea sisestama krediitkaardiandmeid ega sisenema internetipanka. Ostu sooritamiseks tuleb vajutada mTasku maksenupule ja skaneerida arvuti ekraanile ilmuvat QR-koodi nutitelefoniga (sooritades ostu sama nutitelefoniga, jääb skaneerimise samm ära), misjärel avaneb mTasku makse kinnitamise vaade, tasumiseks tuleb vajutada nuppu „Maksan“.


== Installation ==

= Minimum Requirements =

WooCommerce must be installed at least version 3.0.

= Automatic installation =

Automatic installation is the easiest option -- WordPress will handle the file transfer, and you won’t need to leave your web browser. To do an automatic install of WooCommerce, log in to your WordPress dashboard, navigate to the Plugins menu, and click “Add New.”

In the search field type “mtasku,” then click “Search Plugins.” Once you’ve found us,  you can view details about it such as the point release, rating, and description. Most importantly of course, you can install it by! Click “Install Now,” and WordPress will take it from there.

= Manual installation =

Manual installation method requires downloading the mTasku payment for WooCommerce plugin and uploading it to your web server via your favorite FTP application. The WordPress codex contains [instructions on how to do this here](https://wordpress.org/support/article/managing-plugins/#manual-plugin-installation).

= Updating =

Automatic updates should work smoothly, but we still recommend you back up your site.
